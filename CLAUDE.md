\# 项目基本信息

\- \*\*名称\*\*：TommyProject

\- \*\*技术栈\*\*：如：Node.js + Vue 3 + MongoDB

 

\# 目录结构

`C:\Tommyproject\`

├── src/        # 源代码

├── public/     # 静态资源

└── tests/      # 测试文件

 

\# 代码规范

1\. JavaScript/TypeScript 强制使用 ES6+ 语法

2\. 组件命名：PascalCase 风格

3\. 变量命名：camelCase 风格

 

\# 常用命令

\- 启动开发服务器：`npm run dev`

\- 运行测试：`npm test`

\- 构建生产包：`npm run build`
